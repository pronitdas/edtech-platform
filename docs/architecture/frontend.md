# Frontend Architecture

The frontend is a modern React 19 application built with Vite and TypeScript. It uses a component-based architecture with a clear separation of concerns. The state is managed using React Context API, and the UI is built with TailwindCSS.

## Key Features

* **React 19 & Vite**: The latest in frontend technology for a fast and modern development experience.
* **TypeScript**: For type safety and improved code quality.
* **Component-Based Architecture**: For modular and reusable code.
* **React Context API**: For efficient state management.
* **TailwindCSS**: For a utility-first approach to styling.
* **Storybook**: For component documentation and visual testing.
