# Testing

This guide explains how to run the tests for the EdTech Platform.

## Frontend

To run the frontend tests, use one of the following commands:

```bash
# Run all tests
pnpm test

# Run tests with a UI
pnpm test:ui

# Generate a coverage report
pnpm test:coverage
```

## Backend

To run the backend tests, use the following command:

```bash
pytest
```
