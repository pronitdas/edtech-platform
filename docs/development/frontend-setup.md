# Frontend Setup

This guide explains how to set up the frontend for the EdTech Platform.

## Prerequisites

* Node.js and pnpm
* A running instance of the backend API

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

This will start the Vite development server on `http://localhost:5176`.

## Building for Production

```bash
pnpm build
```

This will create a production-ready build in the `dist` directory.
